# Database Schema

## 1. User Collection (`users`)
- `fullName` (String, required)
- `email` (String, required, unique)
- `passwordHash` (String, required)
- `role` (Enum: `student`, `faculty`, `admin`)
- `usn` (String) - Only for students
- `departmentId` (ObjectId, ref: `Department`)
- `cgpa` (Number)
- `skills` (Array of Strings)

## 2. Achievement Collection (`achievements`)
- `studentName` (String)
- `usn` (String)
- `department` (Enum: `CSE`, `ECE`, etc.)
- `achievementTitle` (String)
- `category` (Enum: `Academic`, `Sports`, `Technical`, etc.)
- `description` (String)
- `certificateLink` (String, optional)
- `achievementDate` (Date)
- `status` (Enum: `pending`, `approved`, `rejected`) - Default: `pending`
- `rejectionReason` (String)
- `userId` (ObjectId, ref: `User`)
- `approvedBy` (ObjectId, ref: `User`)

## 3. Department Collection (`departments`)
- `name` (String)
- `code` (String, unique)
- `hod` (ObjectId, ref: `User`)

## 4. Paper Collection (`papers`)
- `title` (String)
- `type` (String)
- `authors` (String)
- `abstract` (String)
- `departmentId` (ObjectId, ref: `Department`)
- `submittedBy` (ObjectId, ref: `User`)
- `status` (Enum: `pending`, `approved`, `rejected`)

## 5. Event Collection (`events`)
- `title` (String)
- `type` (String)
- `organizer` (String)
- `date` (Date)
- `status` (String)

## 6. Portfolio Collection (`portfolios`)
- `studentId` (ObjectId, ref: `User`)
- `bio` (String)
- `headline` (String)
- `careerObjective` (String)
- `skillsSummary` (String)
- `theme` (String)
- `achievements` (Array)
- `papers` (Array)
